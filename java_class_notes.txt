To use the AStar Classes in MATLAB:

1) Define the following Java String objects:
S = Starting Point
X = Obstacle at Coordinate
E = Ending Point
0 (Zero) = Free Space

javaS = java.lang.String('S');
javaX = java.lang.String('X');
javaE = java.lang.String('E');
java0 = java.lang.String('0');

2) Create a 2-Dimensional Java String Array to represent the map:
map = javaArray('java.lang.String', 3, 3);

3) Populate the map with the desired values.

Example:
If we want to make the board:
S 0 0
0 X 0
0 0 E

We need:

map(1, 1) = javaS;
map(1, 2) = java0;
map(1, 3) = java0;

map(2, 1) = java0;
map(2, 2) = javaX;
map(2, 3) = java0;

map(3, 1) = java0;
map(3, 2) = java0;
map(3, 3) = javaE;

4) The A* object requires that the map be represented as a 2D java.util.Vector object.
I wrote an ArrayToVector class that converts a 2D String Array to a 2D String Vector.

atv = ArrayToVector(map);

5) Get the converted Vector map:
realmap = atv.convertArray();

6) Create the A* Object:
astar = AStar(realmap);

7) Call A*:
astar.AStarAlgorithm()
